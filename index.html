2. หน้า "ผู้ป่วย" คืออะไร?

หลังเข้าสู่ระบบสำเร็จ กดเมนู "ผู้ป่วย" (ไอคอน 👥) ที่แถบเมนูบน หรือบาร์ล่างในมือถือ

ท่านจะเห็น เฉพาะรายชื่อผู้ป่วยในหมู่บ้านของท่านเท่านั้น — แต่ละการ์ดจะแสดง: ชื่อ-สกุล, HN, วันตรวจล่าสุด, ค่า HbA1c และ BP ล่าสุด

🟢 🟡 🔴 ความหมายของสีแถบ HbA1c

สีค่า HbA1cหมายความว่า
🟢 เขียว< 7%คุมได้ดี
🟡 เหลือง7 – 8%คุมพอใช้ — ควรติดตาม
🔴 แดง> 8%คุมไม่ได้ — ส่งต่อ จนท. รพ.สต.

🔍 ค้นหา / กรอง

  • ช่องค้นหา — พิมพ์ชื่อ, HN, หรือเลขบัตรประชาชน
  • เรียงลำดับ — เลือกได้ตามชื่อ, HN, HbA1c ล่าสุด, วันตรวจล่าสุด
  • ตัวกรองด่วน — มี 4 ปุ่ม: ทั้งหมด · ยังไม่ตรวจปีนี้ · HbA1c > 8 · BP ไม่คุม
💡 เคล็ดลับ: ใช้ตัวกรอง "ยังไม่ตรวจปีนี้" เพื่อหารายชื่อผู้ป่วยที่ต้องไปตามให้มาตรวจ — เป็นเครื่องมือวางแผนงานที่ดีมากสำหรับ อสม.
คู่มือ อสม. — เพิ่ม/แก้ไขผู้ป่วย หน้า 4

3. การเพิ่มผู้ป่วยใหม่

⚠ สำคัญ: ผู้ป่วยที่ อสม. เพิ่มเข้ามาใหม่จะมีสถานะ รออนุมัติ ต้องรอ admin หรือ จนท. รพ.สต. กดอนุมัติก่อน ผู้ป่วยจึงจะปรากฏในระบบและพร้อมให้บันทึกการเยี่ยม/Lab ได้

เริ่มเพิ่มผู้ป่วยใหม่

  1. กดปุ่ม "+ เพิ่มผู้ป่วย" ที่มุมขวาบนของหน้ารายชื่อผู้ป่วย
    (บนมือถือ — กดไอคอน + เพิ่ม ในแถบล่าง)
  2. หัวฟอร์มจะแสดงป้าย ⏳ ต้องรอ admin อนุมัติ เพื่อให้รู้ว่าข้อมูลนี้จะส่งไปรอการอนุมัติ
  3. กรอกข้อมูลส่วนตัวให้ครบ:
    • เลขบัตรประชาชน (จำเป็น)
    • ชื่อ-สกุล (จำเป็น), เพศ, วันเกิด (อายุจะคำนวณให้อัตโนมัติ)
    • เบอร์โทรศัพท์, ที่อยู่
  4. หมู่บ้าน — ระบบจะล็อกเป็นหมู่ของท่านและไม่ให้แก้ไข (ถ้าหมู่บ้านไม่ตรง ต้องแจ้ง admin แก้สิทธิ์ให้ก่อน)
  5. กรอกข้อมูลการวินิจฉัย: ชนิดเบาหวาน (Type 2 DM), วันที่วินิจฉัย, โรคร่วม (ความดัน/ไขมัน/ไต)
  6. (ทำเลือก) ถ่ายรูปผู้ป่วย — กด "ถ่ายภาพ" จากกล้องมือถือ
  7. ตรวจทานอีกครั้ง — แล้วกดปุ่ม บันทึก
⬇
✅ บันทึกสำเร็จ — รอ Admin อนุมัติ

ข้อมูลผู้ป่วยใหม่ถูกบันทึกแล้ว แต่ต้องรอผู้ดูแลระบบ (admin) อนุมัติก่อน จึงจะเข้าสู่ระบบและใช้บันทึกการเยี่ยม/Lab ได้

4. การแก้ไขข้อมูลผู้ป่วย

การแก้ไขข้อมูลของผู้ป่วยเดิม ไม่ต้องรอการอนุมัติ — บันทึกแล้วเปลี่ยนแปลงทันที (เฉพาะผู้ป่วยในหมู่บ้านของท่านเท่านั้น)

วิธีแก้ไข

  1. ในหน้า ผู้ป่วย กดที่การ์ดของผู้ป่วยที่ต้องการแก้ไข (จะเข้าหน้ารายละเอียด)
  2. มุมขวาบนจะมีปุ่ม "✏ แก้ไข" — กดปุ่มนี้
  3. ปรับเปลี่ยนข้อมูลที่ต้องการ เช่น:
    • เบอร์โทรศัพท์เปลี่ยน
    • ที่อยู่เปลี่ยน
    • HN ที่ รพ.สต. ออกให้ใหม่
    • โรคร่วม (เพิ่ม/ลด HT, DLP, CKD)
    • วันที่วินิจฉัยเบาหวาน (ถ้ารู้แล้ว)
  4. กดปุ่ม บันทึก — ระบบบันทึกทันที

⚠ สิ่งที่ อสม. แก้ไม่ได้

รายการเหตุผล
หมู่บ้านป้องกันการย้ายผู้ป่วยข้ามหมู่บ้านโดยไม่ตั้งใจ — ถ้าย้ายจริงให้แจ้ง จนท. รพ.สต.
ผู้ป่วยจากหมู่อื่นดูได้แต่แก้ไม่ได้ ไอคอนแก้ไขจะไม่ปรากฏ
ลบผู้ป่วยเป็นสิทธิ์ของ admin เท่านั้น — ถ้าผู้ป่วยซ้ำ/ผิด ให้แจ้ง admin

📅 บันทึกการตรวจประจำปี (เท้า/ตา/ฟัน)

  1. เปิดผู้ป่วย → ไปแท็บ "ตรวจประจำปี"
  2. เลือกวันที่ตรวจของแต่ละรายการ (เท้า/ตา/ฟัน)
  3. กด บันทึก ในแต่ละรายการ
💡 ทำไมต้องบันทึก: ระบบจะใช้วันที่นี้คำนวณ KPI "อัตราตรวจเท้า/ตา/ฟัน ประจำปี" ซึ่งเป็นตัวชี้วัด สปสช. ที่ รพ.สต. ต้องรายงานทุกปี

5. การติดตามสถานะ "รออนุมัติ"

หลังจากท่านเพิ่มผู้ป่วยใหม่แล้ว ผู้ป่วยจะปรากฏในรายการของท่าน พร้อมป้ายสีเหลือง ที่บ่งบอกสถานะ

ป้ายหมายความว่าทำอะไรได้
รออนุมัติ เพิ่งบันทึกใหม่ — รอ admin/staff อนุมัติ ติดต่อ admin ให้พิจารณา
approved (ไม่มีป้าย) อนุมัติแล้ว — เข้าระบบเรียบร้อย ใช้งานได้ปกติ
rejected (ผู้ป่วยจะหายไป) admin ปฏิเสธ — ผู้ป่วยถูกซ่อนจากระบบ สอบถาม admin ว่าทำไมถึงปฏิเสธ
📞 ใครคือ admin?
คือ ผู้รับผิดชอบดูแลระบบ — มักเป็นเจ้าหน้าที่ รพ.สต. ที่หัวหน้ามอบหมาย หากไม่แน่ใจให้สอบถามผู้อำนวยการ รพ.สต. หรือเจ้าหน้าที่ฝ่ายข้อมูล

6. ข้อควรระวัง 5 ข้อ

  1. ตรวจเลขบัตรประชาชนให้ถูกต้อง — เพราะระบบใช้เลข ปชช. ตรวจซ้ำ ถ้าใส่ผิด อาจสร้างผู้ป่วยซ้ำได้
  2. ใส่ HN ถ้ามี — HN เป็นเลขทะเบียนของ รพ.สต./โรงพยาบาล จะช่วยเชื่อมข้อมูลกับ Lab
  3. ตรวจทานก่อนกดบันทึก — โดยเฉพาะ ชื่อ-สกุล วันเกิด เพราะแก้ทีหลังก็ยุ่ง
  4. ไม่เปลี่ยนรหัสผ่านให้คนอื่น — แต่ละบัญชีเป็นของบุคคล ห้ามแชร์
  5. ออกจากระบบทุกครั้ง หลังใช้เสร็จ (เมนูชื่อ → ออกจากระบบ) เพื่อความปลอดภัย

7. คำถามที่พบบ่อย (FAQ)

ทำไมเพิ่มผู้ป่วยแล้วยังไม่เห็นในระบบ?
เพราะอยู่ในสถานะ รออนุมัติ — ท่านจะเห็นในรายการของหมู่บ้านท่านพร้อมป้ายสีเหลือง แต่จะยังบันทึกการเยี่ยม/Lab ไม่ได้จนกว่า admin จะกดอนุมัติ
ทำไมแก้ "หมู่บ้าน" ไม่ได้ตอนเพิ่ม/แก้ไขผู้ป่วย?
ระบบล็อกหมู่บ้านเป็นของท่านอัตโนมัติ เพื่อป้องกันการเพิ่ม/แก้ไขผู้ป่วยข้ามหมู่บ้านโดยไม่ตั้งใจ ถ้าผู้ป่วยย้ายจริง — แจ้งเจ้าหน้าที่ รพ.สต. ให้แก้ไขแทน
ใส่ผู้ป่วยซ้ำ จะรู้ได้อย่างไร?
ระบบจะตรวจ HN และเลขบัตรประชาชนซ้ำให้ตอนบันทึก ถ้าซ้ำจะแจ้งเตือน แต่ถ้าจะให้แน่ใจ ลองค้นหาด้วยชื่อหรือเลข ปชช. ก่อนเพิ่มใหม่
เพิ่มผู้ป่วยจากหมู่บ้านอื่นได้ไหม?
ไม่ได้ — อสม. เพิ่มได้เฉพาะหมู่บ้านของตนเอง ถ้าผู้ป่วยอยู่หมู่อื่น ให้บอก อสม. หมู่นั้น หรือ จนท. รพ.สต. แทน
ผู้ป่วยที่ admin ปฏิเสธไป — ขอแก้แล้วส่งใหม่ได้ไหม?
ผู้ป่วยที่ปฏิเสธจะถูกซ่อน ให้เพิ่มเป็นรายการใหม่อีกครั้ง พร้อมแก้ไขข้อมูลตามที่ admin ระบุเหตุผล
ลืมรหัสผ่านทำอย่างไร?
ติดต่อ admin ให้รีเซ็ตรหัสผ่านให้ — admin มีฟังก์ชัน "Reset Password" สำหรับผู้ใช้
ใช้บนมือถือได้ไหม?
ได้ทั้งหมด — ระบบออกแบบให้ใช้กับมือถือสะดวก ทั้ง Android และ iPhone ถ่ายรูปผู้ป่วยจากกล้องในเครื่องได้เลย
ค่า HbA1c, BP กรอกได้ไหม?
ไม่ได้ — การกรอกผล Lab และวัดสัญญาณชีพเป็นหน้าที่ของ จนท. รพ.สต. อสม. แค่ติดตามให้คนไข้ไปตรวจตามนัด
ลบผู้ป่วยที่เพิ่มผิดได้ไหม?
อสม. ลบไม่ได้ — แจ้ง admin ให้ "ปฏิเสธ" หรือ "ปิดการใช้งาน" ผู้ป่วยรายนั้นแทน
ดูรายงาน KPI ของหมู่บ้านได้ไหม?
ได้ — เมนู "รายงาน KPI" ระบบจะกรองให้เห็นเฉพาะหมู่บ้านของท่านอัตโนมัติ
คู่มือ อสม. — เพิ่ม/แก้ไขผู้ป่วย หน้า 8

8. การ์ดข้อมูลด่วน

พิมพ์หน้านี้แยก ตัด แล้วติดที่หน้าโต๊ะทำงาน หรือใส่ในกระเป๋าไว้ใช้

🔐 ขั้นตอนเข้าระบบ

  1. เปิดเว็บแอป
  2. กรอก ชื่อผู้ใช้/เบอร์โทร + รหัสผ่าน
  3. กด เข้าสู่ระบบ

➕ เพิ่มผู้ป่วยใหม่ — 3 ขั้นตอน

  1. เมนู ผู้ป่วย → กด + เพิ่มผู้ป่วย
  2. กรอกข้อมูลให้ครบ → กด บันทึก
  3. รอ admin อนุมัติ (มีป้าย รออนุมัติ)

✏ แก้ไขข้อมูลผู้ป่วย — 3 ขั้นตอน

  1. เมนู ผู้ป่วย → กดที่ผู้ป่วยที่ต้องการ
  2. กดปุ่ม ✏ แก้ไข มุมขวาบน
  3. แก้ไข → กด บันทึก (มีผลทันที)

📅 บันทึกตรวจประจำปี — 3 ขั้นตอน

  1. เปิดผู้ป่วย → แท็บ ตรวจประจำปี
  2. เลือกวันที่ตรวจ (เท้า/ตา/ฟัน)
  3. กด บันทึก ในแต่ละรายการ

⚠ ข้อจำกัดของบทบาท อสม.

  • เห็นและแก้ไขเฉพาะหมู่บ้านของตนเองเท่านั้น
  • ผู้ป่วยใหม่ต้องรอ admin อนุมัติก่อน
  • เปลี่ยนหมู่บ้านของผู้ป่วยไม่ได้
  • ลบผู้ป่วยไม่ได้ (แจ้ง admin)
  • กรอกผล Lab/สัญญาณชีพไม่ได้ (เป็นงาน จนท. รพ.สต.)
